구현한 내용
세 종류의 도형(삼각형, 사각형, 오각형)이 나옵니다.
도형이 사선으로 (좌측 상단 -> 우측 하단 or 우측 상단 -> 좌측 하단) 내려오는 기능
날아오는 도형을 슬라이스 하는 기능(단, 좌측 아래에서 우측 위로 올라오는 대각선만 가능)
바구니가 좌우로 움직이는 기능
마우스로 슬라이스 선 그리는 기능
도형의 경로 출력하는 기능
도형의 출력 모드를 LINE / FILL로 전환하는 기능
날아오는 속도 빨라지는 기능 : + (최대 제한 가속 속도 O)
날아오는 속도 느려지는 기능 : - (최대 제한 감속 속도 O)
프로그램 종료 기능 - q
슬라이스 된 도형이 바구니에 닿았거나 화면 바깥으로 나갈 경우 사라지는 기능
슬라이스 못한 도형이 화면 바깥으로 나가면 반대편으로 나타나는 기능
이 구현되어 있습니다.

구현하지 못한 내용
※ 슬라이스 된 도형이 중력을 적용해 떨어뜨리는 기능
(이 기능은 대신 속력을 고정시킨 대로 y축만 감소시키는 느낌으로 변경되었습니다.)
※ 어느방향으로 슬라이스 하든 도형이 슬라이스 되는 기능
※ 도형이 바구니에 담겨지는 기능
(이 기능은 바구니에 닿으면 사라지는 기능으로 구현했습니다.)
※ 슬라이스 된 도형은 다시 슬라이스 되지 않으며 슬라이스는 쉬운 버전으로 만들어졌습니다.

키보드/마우스 명령어
m :	도형 출력 모드 변경 FILL(Default) / LINE
p:	도형의 경로 출력 (Default : off)
+ :	도형이 날아오는 속도 가속(증가)
- : 	도형의 날아오는 속도 감속(감소)
q :	프로그램 종료

기타 실행 시 알아야할 요소들
슬라이스 판정이 이상할 수도 있습니다. AABB기법을 사용했으나
웬만한 상황에서 좌측 하단에서 우측 상단으로 선을 그었을 때 충돌판정이 발생해 슬라이스 됩니다.
가끔씩 도형이 슬라이스 된거에 비해 많이 나오는 경우가 있습니다.
